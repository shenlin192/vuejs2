# Integrate Bootstrap into Vue 
https://github.com/sejr/vue-bootstrap/

## Set up a template
`
$ npm install -g vue-cli
$ vue init sejr/vue-bootstrap my-project
$ cd my-project
$ npm install
$ npm run dev
`

## Install vuex
`$ npm install --save vuex`

## Install vue-resource
`$ npm install --save vue-resource`
